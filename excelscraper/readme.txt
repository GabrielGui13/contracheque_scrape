# Instruções para Uso do Extrator de Dados de Comprovantes de Contracheque de Pensionistas

## Descrição

Este programa foi criado para ajudar a extrair informações de comprovantes de contracheque de pensionistas de forma automática. Ele pega as planilhas que você fornecer e gera novas planilhas com os dados organizados.

## Como Usar

1. **Preparar Planilhas**: Coloque as planilhas que você quer processar na pasta chamada **planilhas/entrada**.
2. **Executar o Programa**: Clique duas vezes no arquivo .exe para rodar o programa.
3. **Ver Resultado**: As novas planilhas com os dados extraídos aparecerão na pasta **planilhas/saida** com o prefixo "finalizado".
3. **Ver Resultados Defeituosas**: As novas planilhas com os dados que não puderam ser extraídos aparecerão na pasta **planilhas/saida** com o prefixo "erros".

## Formato das Planilhas de Entrada

Para que o programa funcione corretamente, as planilhas que você colocar na pasta **planilhas/entrada** devem ter exatamente as seguintes colunas, com os nomes escritos da forma exata (respeitando maiúsculas, minúsculas, acentos e espaços):

- Matrícula(com o dígito)
- Vínculo
- CPF(do(a) Pensionista
- N.º Pensionista
- Mês
- ano

Certifique-se de que os nomes das colunas estão exatamente assim, senão o programa pode não funcionar corretamente.
